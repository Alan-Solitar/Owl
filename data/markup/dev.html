Owl   For comfortable reading

Hi!

Thanks for choosing Owl.
I started Owl 2 years ago as a quick side project to help save my eyes from those bright-white webpages.
I never thought it would catch on so well, and it's unexpected popularity encouraged me to keep improving it till it was the best "dark background" solution out there.
But as much as I have enjoyed maintaining and upgrading Owl to fit everyone's needs, it's getting increasingly hard for me to keep Owl up-to-date with the people's needs. And that is the reason I want to invite my fellow developers out there to contribute to the add-on we all love.

Code contributions

Owl is completely open source and its source if freely available on Github.
The next milestone I aim to deliver is Owl v3. Based on inputs from the users and some realizations on my side, I hace sketched a basic outline of what the next version would like. Features like smarter CSS application, better URL filtering, red-light shift mode and a new 'High Contrast' theme are on the list. Over that, I would also appreciate suggestions from other devs out there on how Owl can be improved in terms of features and performance.
If you're a developer and would like to help me build the next version of Owl, head up to the Github Issues page and and contribute in whatever way you like (suggestions, code). Look for issues tagged with "Help Wanted" or with milestone v3. Check this tracker bug for a general list of the targets for v3. I'll be looking forward to discussions and pull requests. See you there :)
